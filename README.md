# QWE
native & distro independent package manager.

## project
simple and featured time saver packager, qwe is a abstraction layer that handle packages natively.

## usage
```console
git clone https://github.com/lipe14-ops/qwe.git
cd qwe
make
cd ./dist/qwe /bin
```
